# ✏️ SOBRE

<p>
Aplicativo para ajudar pessoas a lembrarem de cuidar de suas plantas. Os lembretes variam de planta para planta, com frequências diferenciadas entre si e de acordo com o tipo de planta a ser regada.
</p>
